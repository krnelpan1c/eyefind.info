# eyefind.info
Eyefind from GTA V as a usable homepage with support for multiple search engines and custom user defined engines.
  
You can set it as your new tab page using the following extensions:  
Chrome, Edge, Brave, etc: [New Tab Redirect](https://chromewebstore.google.com/detail/new-tab-redirect/icpgjfneehieebagbmdbhnlpiopdcmna)  
Firefox & derrivatives: [New Tab Override](https://addons.mozilla.org/en-US/firefox/addon/new-tab-override/)  

## Changes made
- Real search engine integration
- Custom search engines
- Realtime day of the week
- Randomized Eyefind featured sites
- Autocomplete searchbar based on history
